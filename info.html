El objetivo de este proyecto es comparar el rendimiento y funcionalidad de pyscript y javascript.
Recogido de internet un dataset con 3510 relatos cortos de terror, conocidos como creepypastas
El archivo de origen es un csv de 195019 lineas. Se ha dividido a la mitad
La primera mitad la procesa Pyscript a través de pandas, cargando el archivo .csv a un dataframe.
La segunda mitad es cargada a tráves de un objeto en formato json con javascript
En ambos casos se carga el dataset y se selecciona un registro al azar para mostrar.
Cada dataset ocupa alrededor de 20mb por eso al cargar la página desde un smarthphone tarda un buen rato.
Una vez cargada la página se puede elegir una historia de la lista o especificar el número de historía en la entrada input. En la sección inferior de la página.
Al estár el dataset cargado, la transición entre historias es practicamente instantánea.
